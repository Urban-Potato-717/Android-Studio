- **2개 기둥 = SQLite + ListView/BaseAdapter** → "수업에서 한 발 나아간 핵심 두 가지"로 프레이밍.

## 02장 처음 만드는 안드로이드 애플리케이션

- **Activity = 화면 하나.** `MainActivity` + `activity_main.xml` 쌍, XML 레이아웃 / Java 로직 분리.
- 프로젝트 생성: Empty Activity 템플릿, Minimum SDK 설정, 한글 경로 불가.
- 기본 레이아웃은 `ConstraintLayout`이나 **이 장 실습은 `LinearLayout` 사용**.
- 필수 XML 속성: `layout_width`/`layout_height`(`match_parent`/`wrap_content`), `@+id/...`, `text`, `src`.
- 문자열은 하드코딩 대신 `res/values/strings.xml`에 등록 → `@string/...` 참조.
- Java 핵심 패턴:
  - `extends AppCompatActivity`, `onCreate()` → `super.onCreate()` → `setContentView()`.
  - `findViewById(R.id.xxx)`로 위젯 연결(캐스팅 필요).
  - 이벤트 = 익명 `View.OnClickListener` + `onClick()`.
  - `Toast.makeText(...).show()`, `Intent`로 다른 앱 호출(`ACTION_DIAL`/`ACTION_VIEW`) → `startActivity()`.
- 범위 밖(이 장에서 안 가르침): Fragment, RecyclerView, ViewBinding, 람다 리스너.

## 04장 기본 위젯 익히기

- 모든 화면 요소는 `View` 상속(=위젯), 담는 컨테이너는 `ViewGroup`.
- **공통 속성**: `id`(@+id), `layout_width`/`height`, `background`, `padding`(내부), `layout_margin`(위젯 간), `visibility`(visible/invisible/gone), `enabled`/`clickable`, `rotation`.
- **TextView**: `text`, `textColor`, `textSize`(dp/sp — sp는 폰트), `typeface`, `textStyle`(bold/italic), `singleLine`. Java: `setText()`/`getText()`.
- **Button 클릭 3단계 (시험 빈출)**: ① 변수 선언 ② `findViewById` 대입 ③ `setOnClickListener(new OnClickListener(){ onClick(){} })`.
- **EditText 값 읽기 3단계**: 선언 → 대입 → `getText().toString()`. 변환: `Integer.parseInt()`, `String.valueOf()`.
- **CompoundButton 계열**: `CheckBox`(독립 다중 선택, `setOnCheckedChangeListener`+`onCheckedChanged`), `Switch`/`ToggleButton`, `RadioButton`+`RadioGroup`(하나만 선택, 반드시 그룹으로 묶기).
- **ImageView/ImageButton**: 이미지는 `res/drawable`, XML `@drawable/...`, 속성 `src`/`maxWidth`/`scaleType`.

## 05장 레이아웃 익히기

- 레이아웃 = `ViewGroup` 상속 컨테이너. 5종: `LinearLayout`, `RelativeLayout`, `TableLayout`, `GridLayout`, `FrameLayout`.
- 공통: `orientation`, `gravity`, `padding`, `layout_weight`, `baselineAligned`.
- **LinearLayout**: `orientation`(vertical/horizontal), `gravity`(안쪽 위젯 정렬) vs `layout_gravity`(자신을 부모 어디에 — 구분 주의), 중첩 시 `layout_weight`로 비율(%) 분배.
- **RelativeLayout**: 부모 기준(`layout_alignParentTop` 등 true/false) / 다른 위젯 기준(`layout_below` 등 `@+id/기준위젯`).
- **TableLayout**: `<TableRow>` 개수=행, Row 안 위젯 수=열.
- **GridLayout**: `rowCount`/`columnCount`, 위젯의 `layout_row`/`layout_column`(0부터), `layout_rowSpan`/`columnSpan`, `layout_gravity=fill`.
- **FrameLayout**: 좌상단부터 겹쳐 출력, `foreground`/`foregroundGravity`.
- 채점: `layout_weight` 비율 분배, `gravity` vs `layout_gravity` 구분, RelativeLayout `@+id` 상대 위치가 포인트. ConstraintLayout은 이 장에서 안 다룸.

## 06장 고급 위젯 다루기

> 실제 범위 = **날짜/시간 위젯 + 기타 위젯(ProgressBar·SeekBar·RatingBar·AutoCompleteTextView)** + 앱 아이콘. **ListView/BaseAdapter/Spinner/CheckBox 어댑터 패턴은 이 장에 없음.**

- **날짜/시간**: `AnalogClock`/`DigitalClock`, `Chronometer`(`start()`/`stop()`), `TimePicker`, `DatePicker`/`CalendarView`. RadioButton + `setVisibility(VISIBLE/GONE)`로 토글하는 패턴.
- **AutoCompleteTextView / MultiAutoCompleteTextView**: 자동완성(Multi는 쉼표 구분).
- **ProgressBar / SeekBar**: 진행률 표시 / 터치로 값 조절.
- **RatingBar** (← BookLog 핵심 근거): 별점 표시. `numStars`, `stepSize`, `rating` 값 읽기/설정. 예제="그림 평가 앱"(별점 → SUBMIT → SCORE).
- 앱 아이콘: `res` 우클릭 → New → Image Asset → `res/mipmap`.

## 07장 메뉴와 대화상자

- **옵션 메뉴**: `res/menu/` XML → `onCreateOptionsMenu()`에서 `MenuInflater`로 inflate → `onOptionsItemSelected()`에서 `getItemId()` 분기. (또는 `menu.add()` 코드 방식.)
- **컨텍스트 메뉴**: 롱클릭 팝업. `registerForContextMenu(view)` → `onCreateContextMenu()` → `onContextItemSelected()`.
- **Toast**: `Toast.makeText(context, msg, LENGTH_SHORT/LONG).show()`, `setGravity()`로 위치 변경.
- **AlertDialog (Builder 패턴)** (← BookLog 스포일러 UX 근거):
  - `new AlertDialog.Builder(context).setTitle().setMessage().setPositiveButton("확인", listener).setNegativeButton("취소", listener).show()`.
  - 리스너에 `null` 주면 동작 없음(시험 포인트).
  - 목록 대화상자: `setItems()`(기본), `setSingleChoiceItems()`(라디오), `setMultiChoiceItems()`(체크박스).

## 08장 파일처리

> **이 장은 "파일 입출력"만 다룸. SharedPreferences / SQLite / SQLiteOpenHelper 안 가르침.**

- **내장 메모리**: `openFileOutput(파일명, MODE_PRIVATE)` + `write(byte[])` / `openFileInput()` + `read()`. 경로 `/data/data/패키지/files`.
- **/res/raw (읽기 전용)**: `getResources().openRawResource(R.raw.파일명)` → `InputStream`.
- **SD 카드(외부 저장소)**: `AndroidManifest.xml` 퍼미션 필요, `Environment.getExternalStorage...()`, `File.listFiles()` → `File[]`.
- 실습: 일기장 앱(`DatePicker`+`Calendar`, 파일명 `연_월_일.txt`), 이미지 뷰어(커스텀 `View`+`onDraw()`, `listFiles()`).

## 09장 그래픽과 이미지

- 핵심: **`View` 상속 + `onDraw(Canvas)` 오버라이드**. `Canvas`(drawPoint/Line/Circle) + `Paint`(setColor/setStrokeWidth/setStyle).
- 터치: `onTouchEvent(MotionEvent)` 오버라이드.
- **Bitmap**: `BitmapFactory.decodeResource()` → `canvas.drawBitmap()`. (drawable / SD 카드 이미지 출력.)
- 기하 변환: `rotate()`/`scale()`/`translate()`/`skew()`.
- 마스크/필터: `BlurMaskFilter`, `EmbossMaskFilter`, `ColorMatrix`+`ColorMatrixColorFilter`.
- 메모: 이 장은 저수준 직접 그리기 중심 — BookLog의 `ImageView`+`Covers`/`ImageLoader` 표지 로딩과는 별개 영역.
